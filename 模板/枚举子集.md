
# 枚举子集


#### 利用二进制枚举子集
* 二进制枚举n的全部子集
```C++
void SubSet(int n) {

    for(int S = 0; S < (1 << n); S++) {
        //S是n的一个子集
        for(int i = 0; i < n; i ++) {
            int x = (S>>i)&1; // 输出第i个元素
            cout << x;
        }
        cout << endl;

    }

}
```
